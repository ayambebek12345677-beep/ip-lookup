# IP Lookup

Look up your public IP address, location, ISP, and timezone information.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Open Lookup** (or Skip).

**Step 3:** Your public IP address is detected automatically along with location details.

**Step 4:** View info cards: Country, City, Region, ISP, Timezone, Coordinates, Postal, ASN.

**Step 5:** Enter a different IP address in the search field to look up its info.

**Step 6:** Click **Lookup** to fetch the data.

## Features
- Auto-detect your public IP
- City, country, region, postal code
- ISP and ASN information
- Timezone detection
- Geographic coordinates
- Search any IP address
- Uses ipapi.co free API

## Tech
Fetch API + ipapi.co + HTML + CSS + JavaScript
